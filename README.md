## Executive Summary  
Customer churn is a critical challenge in the banking industry, directly affecting revenue and profitability. This project analyzes a retail bankâ€™s customer data to uncover churn drivers, highlight key performance indicators, and provide actionable insights for improving customer retention.

## Business Objectives  
- Measure **churn and retention rates** across the customer base.  
- Profile **churned vs. retained customers** using demographics and satisfaction data.  
- Identify **key churn drivers** such as satisfaction, tenure, and age.  
- Build an **interactive dashboard** to support real-time business decisions.
